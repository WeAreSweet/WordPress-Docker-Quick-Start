# WeAreSweet WordPress Starter Kit

## Prerequisites

* Docker
* Docker Compose
* Have WAS Traefix Proxy installed and running.

## Quick install

Run the install script:
```
./install.sh
```

Once the install script has completed, make sure you update this README.md file to match your project.
